Test Scenarios (Phase 1)

🔄 Scenario 1: Page Refresh Protection
Tests if page refresh attempts are blocked. Try pressing F5, Ctrl+R, or Cmd+R. The browser will show a confirmation dialog and the event will be logged.
⬅️ Scenario 2: Back/Forward Blocking
Tests if browser back/forward navigation is blocked. Try clicking the browser's back button or pressing Alt+Left Arrow. Navigation will be prevented.
📋 Scenario 3: Copy-Paste Prevention
Tests if copy, cut, and paste are blocked. Try Ctrl+C, Ctrl+V, or Cmd+C in the textarea below. Copy attempts will be blocked and logged.
🖱️ Scenario 4: Right-Click Disabling
Tests if right-click context menu is disabled. Try right-clicking anywhere on the page. Right-click will be prevented and logged.
🔀 Scenario 5: Tab Switching Detection
Tests if tab switching is detected and logged. Try switching to another browser tab and coming back. The system will detect focus loss and log the event.
